## 使用方法

### 1. 获取弹数列表

下载 `sets.json` 获取所有可用弹数信息：

```json
[
  {
    "id": "28",
    "name": "炽热极限 第三弹",
    "version": "1.0.0",
    "filename": "炽热极限_第三弹_v1.0.0.zip",
    "cardCount": 48,
    "size": "26.1MB",
    "updatedAt": "2024-03-19"
  }
]
```

### 2. 版本对比（iOS APP 示例）

```swift
// 本地保存的版本
let localVersion = UserDefaults.standard.string(forKey: "set_28_version") ?? "0.0.0"

// 从 sets.json 获取最新版本
let latestVersion = setInfo.version

// 版本比较
if localVersion != latestVersion {
    // 需要更新下载
    downloadSet(setInfo)
}
```

### 3. 下载单个弹数数据

根据需要下载对应的 `.zip` 文件，每个压缩包约 25-30MB。

### 4. 解压使用

解压后包含：
- `cards.json` - 该弹所有卡片数据（含 `version` 字段）
- `images/cards/` - 卡片图片
- `images/attributes/` - 属性图标
- `images/skills/` - 技能图标

## 数据统计

- **总弹数**: 8 弹

- **平均每弹**: 48 张卡片

## 版本号规则

- **初始版本**: `1.0.0`
- **数据修正**: 修订号 +1 (如 `1.0.1`)
- **新增卡片/图片**: 次版本号 +1 (如 `1.1.0`)
- **重大结构调整**: 主版本号 +1 (如 `2.0.0`)

## 更新策略

- 每弹独立版本号，可单独更新
- APP 通过对比版本号判断是否需要更新
- 新增弹数时只需添加新的压缩包
