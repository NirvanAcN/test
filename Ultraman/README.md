## 使用方法

### 1. 获取弹数列表

下载 `sets.json` 获取所有可用弹数信息：

```json
[
  {
    "id": "28",
    "name": "炽热极限 第三弹",
    "filename": "炽热极限_第三弹.zip",
    "cardCount": 48,
    "size": "28MB"
  }
]
```

### 2. 下载单个弹数数据

根据需要下载对应的 `.zip` 文件，每个压缩包约 25-30MB。

### 3. 解压使用

解压后包含：
- `cards.json` - 该弹所有卡片数据
- `images/cards/` - 卡片图片
- `images/attributes/` - 属性图标
- `images/skills/` - 技能图标

- 无版本号，直接覆盖更新
- 每弹独立，可单独更新
- 新增弹数时只需添加新的压缩包
