# 激斗流星 第一弹

## 卡片数量
48 张

## 文件结构
```
激斗流星_第一弹/
├── cards.json          # 卡片数据
├── README.md           # 本文件
└── images/
    ├── cards/          # 卡片图片
    ├── attributes/     # 属性图标
    └── skills/         # 技能图标
```

## 使用说明
1. 下载此压缩包
2. 解压到 APP 数据目录
3. 加载 cards.json 获取卡片数据
4. 图片路径为相对于 cards.json 的相对路径

## 数据来源
奥特曼光辉共斗官方数据
